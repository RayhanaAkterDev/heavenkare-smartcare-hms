# HeavenKare SmartCare – AI Assisted Hospital Management System

**HeavenKare SmartCare** is a modern web-based **Hospital Management System (HMS)** designed to streamline hospital operations and improve patient interaction through a digital platform.

The system provides an integrated solution for managing hospital services such as doctor information, appointment scheduling, patient records, and administrative operations. It also introduces **AI-assisted patient support features** to enhance accessibility and decision support.

This project is developed as the **Final Year Project** for the CSE program.

---

## Project Structure

The system follows a **modern full-stack architecture**:

Frontend  → React  
Backend   → Laravel API  
Database  → MySQL  

- **Frontend:** Public hospital website and patient interface  
- **Backend:** API and system logic  
- **Database:** Storage of patient, doctor, and appointment data  

---

## Core Modules

The system consists of the following main modules:

### Public Website

- Hospital homepage
- Services and specialties
- Doctor listing
- Appointment booking
- Contact and hospital information

### Patient System

- Patient registration and login
- Appointment booking
- Appointment history

### Doctor Management

- Doctor profile management
- Consultation schedules
- Patient appointment tracking

### Administrative System

- Patient record management
- Appointment control
- Doctor and staff management

---

### AI Assisted Features (Planned)

- AI-based symptom suggestion
- Basic patient support assistant
- Smart appointment assistance

These features aim to demonstrate the integration of **AI support tools in healthcare systems**.
